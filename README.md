# Kanbanboard

## Installation

Use the package manager NPM  to install, together inside /client as well. 

```bash
npm install
```
And to start the app, Run:

```bash
npm start
```

To seed the database, Run:

```bash
npm run seed
```

### Extra Features:
1. Simple backend Database

### Technologies:
1. React framework  
2. MongoDB Database
3. RestFul APi by using Express and Node.js
